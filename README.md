A crop recommendation system that takes soil & climate inputs and generates intelligent farming suggestions using ML models + LLM-powered reports.  
`Python` `scikit-learn` `Streamlit` `LLM`

### 🧪 PromptCraft
A Streamlit-based LLM prompt testing tool — compare, refine, and evaluate prompts interactively.  
`Python` `Streamlit` `LLM APIs` `Prompt Engineering`

### 📈 Demand Forecast Regression
Compared Linear Regression, Random Forest & Gradient Boosting on a Kaggle dataset. Key insight: applying `np.log1p` to the target variable boosted Random Forest R² to ~0.755.  
`Python` `scikit-learn` `Pandas` `Streamlit`
